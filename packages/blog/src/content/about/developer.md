프로덕트 엔지니어입니다. 사용자가 목표를 쉽게 달성할 수 있는 제품을 만듭니다.

4년간 프론트엔드 개발자로서 사용자 인터페이스를 만들어왔습니다. React, SvelteKit으로 SaaS 제품을 개발하며 로그 분석, 세션 리플레이 등 관측 도구를 활용해 사용자를 이해하고, 디자이너와 치열하게 논의하며 최적의 경험을 고민해왔습니다.

현재는 [LLM Agent 플랫폼](https://coxwave.com)에서 Agent 시스템을 개발하고 있습니다. 사용자 경험을 AI로 확장하는 일에 관심이 생겨 이 영역으로 넘어왔고, Agent 아키텍처부터 LLM 앱의 프론트엔드까지 풀스택으로 제품을 만들어가고 있습니다.

빌드 속도 개선, 모니터링 도구 도입, E2E 테스트 관리, 의존성 관리, 패키지 배포 등 다양한 경험을 쌓았습니다. Nest.js, Express, Docker, PostgreSQL, Redis, MongoDB에 경험이 있습니다.

읽기 쉬운 코드, 수정하거나 제거하기에 편한 코드를 작성합니다. 작은 모듈을 쉽게 합성할 수 있는 구조를 추구하고, 정적 분석 도구로 일관된 품질을 유지합니다.

### 히스토리

[대학 시절 촬영 공간 대여 사업](https://baek.co)을 운영한 경험을 바탕으로 스타트업 액셀러레이터에서 인턴을 하며 제품 개발에 대한 열정을 키웠습니다. 데이터 분석 과정에서 처음 프로그래밍을 접했고, 사용자 인터페이스 개발에 매력을 느껴 프론트엔드 개발자로 커리어를 시작했습니다.

현재는 AI가 실제로 사용자의 문제를 해결하는 제품을 만드는 일에 집중하고 있습니다.
