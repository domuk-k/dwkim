# dwkim

터미널에서 바로 만나는 개발자 프로필 & AI 어시스턴트

## 설치

```bash
npx dwkim
```

전역 설치를 원한다면:
```bash
npm install -g dwkim
```

## 사용법

```bash
dwkim           # 프로필 + AI 채팅
dwkim profile   # 프로필만 보기
dwkim help      # 도움말
```

### 채팅 명령어

```
/help     도움말
/status   서버 상태 확인
/clear    대화 초기화
```
